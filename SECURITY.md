# Security Policy

## Supported Versions
Security updates are provided on the `main` branch until the next stable release.

## Reporting a Vulnerability
Please email security@netguard.app with detailed information. We aim to respond within 72 hours.

When reporting, please include:
- Affected components and versions
- Steps to reproduce
- Potential impact
- Suggested remediation or workaround

We follow responsible disclosure and will work with you to coordinate a fix and public advisory.
